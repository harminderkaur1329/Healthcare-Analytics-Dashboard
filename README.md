# 🏥 Healthcare Analytics Dashboard using Microsoft Power BI

## 📌 Project Overview

This project is an interactive Healthcare Analytics Dashboard developed using Microsoft Power BI. It provides comprehensive insights into hospital operations, patient demographics, financial performance, medical conditions, and hospital efficiency through interactive visualizations.

The dashboard enables healthcare professionals and decision-makers to monitor key performance indicators (KPIs), identify trends, and support data-driven decision-making.

---

## 🎯 Objectives

- Analyze hospital performance and patient data.
- Monitor healthcare financial metrics.
- Understand patient demographics.
- Explore medical condition trends.
- Evaluate hospital operational efficiency.
- Enable interactive analysis using filters and slicers.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Excel / CSV Dataset
